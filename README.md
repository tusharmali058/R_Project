## Dataset

The dataset used in this project is too large to be included in this GitHub repository.

Please download the dataset from the link below and place the `city_day.csv` file inside the project's `data/` folder before running the R script.

**Dataset Source:**  
https://www.kaggle.com/datasets/rohanrao/air-quality-data-in-india

**Required File:**  
- `city_day.csv`
